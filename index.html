Deployment

If you want to deploy the client or the server locally on your own machine, this section is for you. Otherwise you can skip to "Usage in Kekkonen"

Server

Download the master branch from the link up top, or clone the repo: 


git clone https://github.com/tietokilta-saato/tikplay.git
git checkout master

Client locally

Download the client branch from the link up top, or clone the repo: 


git clone https://github.com/tietokilta-saato/tikplay.git
git checkout client

Client also has tags (f.ex. client-v1.0.0) you can checkout to, if you want to use stable and surely working versions.
